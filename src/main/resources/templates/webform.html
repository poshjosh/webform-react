Webform + React - Demo Application

This application demonstrates the versatility of Webforms with a React based front end. Using Webforms, forms are automatically generated for your web application based on annotations defined on your domain objects.

Some annotations used are those in the:
  • javax.validation.constraints package for example: javax.validation.constraints.NotNull will make a field required
  • javax.persistence package for example: @javax.persistence.Column(nullable = false) will make a field optional

There are four (4) CRUD actions supported by Webform

  • Create with endpoint /create
  • Read with endpoint /read
  • Update with endpoint /update
  • Delete with endpoint /delete

To create a model object named Employee you enter: /create/employee and Webform automatically generates a form for the CRUD action you requested (create) based on annotations you defined on the Employee DTO.

This sample application has 3 models, click any of the following links, to open a form for Create:
  • Blog
  • Post
  • Tag
